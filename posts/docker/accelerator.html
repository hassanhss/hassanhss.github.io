阿里云加速器 #

使用Docker时需要首先下载一个官方镜像，例如mysql、wordpress。然而由于网络原因，下载一个Docker官方镜像可能会需要很长的时间，甚至下载失败。为此，阿里云容器镜像服务ACR提供了官方的镜像站点，从而加速官方镜像的下载。

注册阿里云账号 #

首先注册一个阿里云账号，登录地址。在左侧导航栏选择镜像工具 > 镜像加速器，在镜像加速器页面获取镜像加速地址。

在这里你会看到一个专属自己的镜像加速器地址。

配置镜像加速器 #

在不同的操作系统下，配置加速器的方式略有不同，下文将介绍主要操作系统的配置方法。

当Docker版本较新时 #

当载安装的Docker Version不低于1.10时，建议通过daemon config进行配置。使用配置文件/etc/docker/daemon.json（没有时新建该文件）。

{
    "registry-mirrors": ["<镜像加速器地址>"]
}

然后重启Docker Daemon。
